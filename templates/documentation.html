Catalog


Description

Returns a JSON object containing all categories and the items within each category.

URI

{{url_for('apiCatalog')}}

Structure

"results": The response body.
{
- "categories": A list containing all categories.
- [
- - {
- - - "id": The category's id.
- - - "name": The category's name.
- - - "items": A list containing all of this category's items.
- - - [
- - - - {
- - - - - "id": The item's id.
- - - - - "name": The item's name.
- - - - - "description": The item's description.
- - - - - "image_url": The url for the item's image.
- - - - - "category_id": The id for the item's category.
- - - - - "user_id": The id for user that created the item.
- - - - }
- - - ]
- - }
- ]
}
